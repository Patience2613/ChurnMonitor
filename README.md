# ChurnMonitor

**ChurnMonitor** is a professional web application for predicting customer churn and analyzing customer retention trends. It combines a **FastAPI backend** for machine learning predictions and **PostgreSQL** storage with a **React frontend** for interactive dashboards, tables, and visualizations.

---

## **Features**

- Predict customer churn using a trained machine learning model.
- Store and manage customer data in PostgreSQL.
- Interactive dashboard with charts showing churn analytics.
- Edit and delete customer records from the dashboard.
- User-friendly UI with professional color-coded visuals.

---

## **Tech Stack**

- **Backend:** Python, FastAPI, SQLAlchemy, Joblib, PostgreSQL
- **Frontend:** React, Recharts, Axios
- **Machine Learning:** Scikit-learn
- **Deployment:** Render
